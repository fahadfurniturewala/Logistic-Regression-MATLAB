Instructions to run the function logistic_regression in MATLAB.
The name of the function is logistic_regression.
In the MATLAB command window enter the following commands.


1.type  " logistic_regression pendigits_training.txt 1 pendigits_testing.txt " in the command window in Matlab
2.Press enter



The first argument is the training file name which can be changed as and when needed.
The second argument is the degree which can be either 1 or 2.
The first argument is the testing file name which can be changed as and when needed.

The other possible running combinatins are:

logistic_regression pendigits_training.txt 2 pendigits_testing.txt
